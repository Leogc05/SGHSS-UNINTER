Conformidade LGPD

O SGHSS segue integralmente a Lei Geral de Proteção de Dados (Lei 13.709/2018), aplicando princípios como finalidade, necessidade, transparência e segurança.

  • Coleta apenas dados essenciais para o atendimento médico.
  • Consentimento explícito para uso de dados sensíveis.
  • Criptografia aplicada em banco de dados, APIs e transmissão.
  • Logs detalhados de todas as operações do sistema.
  • Controle de acesso baseado em perfis (Paciente, Médico, ADM).
  • Direito de acesso, correção e eliminação dos dados pelo paciente.
  • Backups diários e replicação segura para prevenção de perda de dados.
  • Retenção mínima obrigatória de prontuário médico: 20 anos.
